*************
Installation
*************

.. _PyPI: 
.. _source: https://pypi.org/project/dataset-hub/

**Install** the package by the following command from PyPI_:

.. code-block:: bash
    
    # Python 3.9+
    pip install dataset-hub

If you run into installation issues, please write `here <https://github.com/GetDataset/dataset-hub/discussions/64>`_.
